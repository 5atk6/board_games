# Board Games

## How to build
```
git clone https://github.com/5atk6/board_games.git
cd board_games
mkdir build && cd build
cmake ..
make
```

## How to play
```
cd build
./app
```
Type place like 1 2.
First number represents y, second one represents x.
If you make a line made by 5 stones, you are winner.
